# ANTOINE-SEMENYO-DATA-ANALYSIS-PROJECT-
# INTRODUCTION
This is a Python Project that shows analysis of data gotten from various Premier League Data files in Fbref(A Sports statisitcs/data platform).
# SECTION A: GOAL
The goal of this project is  to make a recruitment style  data analysis report on Antoine Semenyo to possibly simulate the processes which took place in Manchester City's Data Department to determine the viablity of his signing from a DATA DRIVEN STANDPOINT
# SECTION B: SKILLS DEMOSTRATED DURING THIS PROJECT
1.Data Sourcing 2.Data Transformation 3.Data Cleaning 4.Data Visualization
# SECTION C : PROBLEM STATEMENT
This study aims to compare the performances of Antoine Semenyo statistically against other attackers in the English Premier League in terms of key attacking metrics like goals,assists and carries into the penalty area amongst others.It also compares these offensive metrics across time to see if consistency indeed exists in his attacking metrics in order to validate or invalidate the need of his signing to   Manchester City from a DATA- DRIVEN STANDPOINT.
